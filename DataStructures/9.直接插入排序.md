# 直接插入排序（属于内部排序）

**插入排序**：

```
1.直接插入排序
2.希尔排序
```

**插入排序**思想：

```
	将数组分为有序表和无序表，每次从无序表中拿出一个数据到有序表中遍历比较，找到插入位置即可。当遍历完无序表就结束了。
```

代码简介：

```
1.temp 临时变量（用来存储待插入的值）
2.index 待插入的位置（用来存储待插入的位置）
	使用双重循环来实现插入排序，外层for循环是遍历数组种的无序表（从1到arr.length - 1)，内循环是有序列表，每次外层循环完成，数组的有序表长度+1，完成一个元素的位置确定。内层for循环是从无序列表头位置n开始循环到找到合适位置m为止（n->m n>m 倒序循环 n在数组右侧 ，m在数组左侧）找到合适位置即可插入元素。
```

代码：

```
	public static void insertSort(int arr[]) {
		int temp;
		int index;
		for (int i = 1; i < arr.length; i++) {
			temp = arr[i];
			index = i;
			for (int j = i - 1;j >= 0 && temp <= arr[j]  ; j--) {
				arr[j + 1] =arr[j];
				index = j;

			}
			arr[index] = temp;

		}

	}
```
